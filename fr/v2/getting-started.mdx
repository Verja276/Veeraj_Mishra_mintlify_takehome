# Démarrage avec Salsa as a Service v2.0

Bienvenue sur v2.0 ! Ce guide vous aidera à configurer votre compte et à exploiter les nouvelles fonctionnalités alimentées par IA, capteurs IoT et intégrations avancées.

## Prérequis

Avant de commencer, assurez-vous d'avoir :

- Une adresse e-mail professionnelle valide
- Accès aux systèmes de votre restaurant (POS, inventaire, etc.)
- Connaissance de base de vos opérations actuelles de salsa
- Accès administrateur pour configurer les intégrations
- **Nouveau dans v2.0** : Matériel de capteurs IoT optionnel pour surveillance qualité

## Configuration du compte

### Étape 1 : Création de compte avancée

1. Visitez notre [page d'inscription](https://app.salsaasaservice.com/signup)
2. Choisissez votre méthode d'authentification :
   - E-mail/mot de passe avec 2FA
   - SSO (Google, Microsoft, Okta)
   - SAML 2.0 pour entreprise
3. Vérifiez votre e-mail ou fournisseur SSO
4. Complétez la vérification d'entreprise améliorée

<Note>
  **Note:**
  **Sécurité renforcée** : v2.0 supporte l'authentification biométrique, clés de sécurité matérielles et options de connexion sans mot de passe.
</Note>


## Configuration des fonctionnalités IA

### Configuration de prévision de demande

Activez la prévision de demande alimentée par IA pour prédire les besoins en salsa 7-30 jours à l'avance :


1. **Activer les prévisions**
   Allez dans **Fonctionnalités IA > Prévision de demande** et activez la fonctionnalité.

2. **Connecter les sources de données**
   Liez votre système POS, système de réservation et toutes sources de données externes.

3. **Import de données historiques**
   Importez au moins 90 jours de données de ventes historiques pour une précision optimale.

4. **Définir les paramètres**
   Configurez l'horizon de prévision (7, 14 ou 30 jours) et les seuils d'alerte.



<Note>
  **Note:**
  **Précision des prévisions** : L'IA atteint une précision de 70-80% avec 90 jours de données et s'améliore à 95%+ après 6 mois.
</Note>

1. **Sélection du matériel**
   Choisissez des capteurs compatibles de notre catalogue certifié.

2. **Installation physique**
   Suivez le guide d'installation. Notre application mobile inclut la visualisation AR pour le placement.

## Première production avec IA


1. **Consulter les prévisions IA**
   Vérifiez la prévision de demande dans **Tableau de bord > Prévisions**.

2. **Démarrer la production intelligente**
   L'IA recommande la taille de lot et le timing optimaux.

3. **Surveillance en temps réel**
   Les capteurs IoT suivent automatiquement la qualité de production.

4. **Compléter et apprendre**
   Complétez le lot. L'IA apprend pour améliorer les prévisions futures.
